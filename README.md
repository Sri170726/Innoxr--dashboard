# InnoXR Device Management Dashboard

## Project Overview

This project is a React-based web application developed as part of the InnoXR Software Development Internship assignment.

The application provides a simple interface for managing HoloInteract devices through three main screens:

* Login Page
* Device Dashboard
* Analytics Dashboard

---

## Features

### Login Page

* Email input
* Password input
* Login button
* Navigation to Dashboard

### Device Dashboard

* Displays device name
* Displays location
* Online/Offline status
* Last active time

### Analytics Dashboard

* Total Devices
* Active Devices
* Interactions
* Campaign Performance

## Installation

Clone the repository

```
git clone <repository-url>
```

Navigate into the project

```
cd innoxr-dashboard
```

Install dependencies

```
npm install
```

Run the application

```
npm run dev
```
